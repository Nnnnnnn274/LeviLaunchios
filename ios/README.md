# LeviLauncher iOS

iOS port of LeviLauncher - a mod loader / content manager for Minecraft: Bedrock Edition.

## Architecture

This is a **dylib** that gets injected into Minecraft: Bedrock Edition (iOS) using idevicekit / TrollStore. It provides:

- In-game mod menu overlay (floating button)
- Microsoft/Xbox account management
- World, resource pack, server, and screenshot management
- CurseForge content browser
- Built-in mods (zoom, FPS counter, snaplook)
- LeviLamina-compatible mod loading via C++ preloader

## Prerequisites

- macOS with Xcode 15+
- Jailbroken iOS device or TrollStore
- Decrypted Minecraft IPA

### Tools

```bash
brew install ldid optool xcodegen
brew install ideviceinstaller  # for non-TrollStore installation
```

## Build & Inject

```bash
# 1. Setup
cd ios
chmod +x setup.sh inject.sh
./setup.sh

# 2. Build the dylib
xcodebuild -project LeviLauncher.xcodeproj \
           -scheme LeviLauncher \
           -configuration Release \
           -sdk iphoneos \
           CODE_SIGNING_ALLOWED=NO

# 3. Inject into Minecraft
./inject.sh path/to/Minecraft.ipa

# Or manually with TrollStore:
# - Decrypt Minecraft IPA
# - Inject LeviLauncher.framework into Payload/*.app/Frameworks/
# - Add load command with optool
# - Re-sign with ldid
# - Package as .ipa and open in TrollStore
```

## Manual Injection (TrollStore)

1. Get a decrypted Minecraft IPA
2. `unzip Minecraft.ipa -d Payload/`
3. Copy `LeviLauncher.framework` to `Payload/*.app/Frameworks/`
4. `optool install -c load -p @executable_path/Frameworks/LeviLauncher.framework/LeviLauncher -t Payload/*.app/Minecraft`
5. `ldid -S Payload/*.app`
6. `zip -qr LeviLauncher-Minecraft.ipa Payload/`
7. Open in TrollStore

## Structure

```
ios/
├── LeviLauncher.xcodeproj/   # Xcode project (generated by xcodegen)
├── project.yml               # XcodeGen spec
├── setup.sh                  # Environment setup
├── inject.sh                 # Build + inject into IPA
├── LeviLauncher/
│   ├── EntryPoint.m          # C constructor (runs on dlopen)
│   ├── LauncherEntry.swift   # Swift initialization
│   ├── PreloaderHook.swift   # Minecraft ObjC runtime hooks
│   ├── Core/                 # Business logic (auth, content, NBT, etc.)
│   ├── UI/                   # In-game overlay UIKit views
│   ├── Native/               # C++ preloader + built-in mods
│   └── Resources/            # Asset catalog, colors
```
